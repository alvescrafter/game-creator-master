# Skill: Entertainment Game Generation

> Game Creator AI — Generate game specs, mechanics, and prompt details for arcade, platformer, shooter, strategy, and other entertainment game types.

## Overview

**App:** Game Creator AI
**URL:** https://aigamecreator.netlify.app/
**Category:** Entertainment
**Price:** Free ($0)

## What It Does

Game Creator AI is a browser-based AI tool that generates entertainment game specifications. Users describe a game concept, and the AI produces a complete game spec with:

- Game mechanics and rules
- Art style direction
- Tone and narrative elements
- Gameplay flow
- Prompt-driven customization

## Who It's For

- Indie game developers exploring concepts
- Hobbyists prototyping game ideas
- Game jam participants
- Rapid prototypers
- Creative writers exploring interactive fiction

## How to Use

1. Open https://aigamecreator.netlify.app/ in any modern browser
2. Configure an AI provider:
   - Enter an API key for OpenAI, Anthropic, or Google
   - Or connect a local model via Ollama or LM Studio
   - Or use a custom OpenAI-compatible endpoint
3. Describe the game:
   - Genre (arcade, platformer, shooter, strategy, etc.)
   - Art style and tone
   - Core mechanics
   - Target audience
   - Any specific requirements
4. The AI generates a complete game spec with mechanics, art direction, and prompt details

## Supported AI Providers

- OpenAI (GPT models)
- Anthropic (Claude models)
- Google (Gemini models)
- Ollama (local, offline)
- LM Studio (local, offline)
- Custom OpenAI-compatible endpoints

## Capabilities

- `game_spec_generation` — Generate complete game specifications
- `arcade_game_creation` — Arcade-style game specs
- `platformer_game_creation` — Platformer game specs
- `shooter_game_creation` — Shooter game specs
- `strategy_game_creation` — Strategy game specs
- `art_style_customization` — Fine-tune visual direction
- `mechanic_fine_tuning` — Adjust gameplay mechanics

## Limitations

- Requires user-provided API key or local model (BYOK)
- Generates game specifications and concepts, not playable game files
- No backend — runs entirely client-side
- No user accounts or data persistence beyond browser localStorage
