# Skill: Educational Game Generation

> Game Creator Learn — Generate learning games with lesson structure, quizzes, and classroom-ready themes.

## Overview

**App:** Game Creator Learn
**URL:** https://aigamecreatorlearn.netlify.app/
**Category:** Education
**Price:** Free ($0)

## What It Does

Game Creator Learn is a browser-based AI tool that generates educational games. Users describe a subject, topic, and target audience, and the AI produces a complete learning game with:

- Lesson structure and learning objectives
- Quiz questions and answers
- Classroom-ready formatting and themes
- Age-appropriate content

## Who It's For

- Teachers creating classroom activities
- Tutors building practice materials
- Students studying through gamified content
- Homeschooling parents
- Educational content creators

## How to Use

1. Open https://aigamecreatorlearn.netlify.app/ in any modern browser
2. Configure an AI provider:
   - Enter an API key for OpenAI, Anthropic, or Google
   - Or connect a local model via Ollama or LM Studio
   - Or use a custom OpenAI-compatible endpoint
3. Describe the learning game:
   - Subject (maths, science, language, history, etc.)
   - Topic within the subject
   - Target age group / grade level
   - Difficulty level
   - Any specific requirements
4. The AI generates a complete educational game spec with lesson structure and quizzes

## Supported AI Providers

- OpenAI (GPT models)
- Anthropic (Claude models)
- Google (Gemini models)
- Ollama (local, offline)
- LM Studio (local, offline)
- Custom OpenAI-compatible endpoints

## Capabilities

- `educational_game_generation` — Generate complete learning game specs
- `lesson_structure_creation` — Create structured lesson plans
- `quiz_generation` — Generate quiz questions and answers
- `classroom_ready_themes` — Apply classroom-appropriate formatting

## Limitations

- Requires user-provided API key or local model (BYOK)
- Generates game specifications and concepts, not playable game files
- No backend — runs entirely client-side
- No user accounts or data persistence beyond browser localStorage
